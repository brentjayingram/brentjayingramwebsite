# Brent Ingram - Personal Website

A clean, professional portfolio website showcasing my experience as a Cloud & Automation Engineer.

## Features

- **About Page**: Personal introduction and background
- **Resume**: Detailed work experience and skills
- **Blog**: Technical articles and insights
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Static site (no framework dependencies)

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required

## Structure

```
├── assets/
│   ├── style.css           # Main stylesheet
│   ├── main.js            # JavaScript functionality
│   └── Profile Picture Fall Small.jpg
├── blog/                  # Blog posts in Markdown
├── about.html            # About page
├── resume.html           # Resume page
├── blog-index.html       # Blog listing
└── index.html           # Homepage
```
